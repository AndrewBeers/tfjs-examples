Game Simulation

Click "Simulate Game" to run one play of the game. Three numbers will be randomly selected for each player. The "win" status will indicate whether player one's numbers beat the opponent's numbers.

Rules:
  • Any triple beats any double.
  • Any double beats any single.
  • If both players have a triple, the higher triple wins.
  • If both players have a double, the higher double wins.
  • If neither player has a double, the player with the highest individual number wins.
  • Ties go to the opponenet.
